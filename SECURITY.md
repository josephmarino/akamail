## I found a security issue, how do I report it?

If you found a security issue such as a vulnerability, please share your finding with us by using GitHub's [report a vulnerability](https://github.com/josephmarino/akamail/security/advisories/new) feature. Please provide clear instructions on how to reproduce the vulnerability. If you are capable of writing a fix for the vulnerability, please feel free to write a fix and contribute the code to AKA Mail's repository.

If you found any security issues in any third-party libraries that AKA Mail is dependent upon, then please reach out to the team who maintains that third-party library.
